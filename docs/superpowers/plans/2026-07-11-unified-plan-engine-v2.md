**Goal:** One shared engine produces every trade plan the bot emits ‚Äî badge-stamped with real out-of-sample win-rate pedigree (‚úÖ VALIDATED ‚â•80% / ‚öÝÔ∏è WEAK), hybrid scale-out exits, and an intraday plan-lifecycle manager ‚Äî so live behavior equals backtested behavior by construction.

**Architecture:** New `swingbot/core/plan_engine.py` (TradePlanV2 + sizing + exit simulator) absorbs the duplicated sizing in `backtest._trade_plan_at` / `trade_plan.py` / `levels.build_scenarios`; a scenario backtest brings the confluence scan under the same train/validate discipline; `plan_manager.py` evolves the 60s trade monitor into a PENDING‚ÜíACTIVE‚ÜíPARTIAL‚ÜíCLOSED lifecycle; feature flags + shadow mode gate the cutover; a final rescue phase re-attempts the 5 failing strategies with pre-registered hypotheses.

**Tech Stack:** Python 3.11+, discord.py 2.7.1, pandas 2.3.3, numpy, yfinance 0.2.66, mplfinance, pytest ‚â•8. JSON persistence under `data/`. No new dependencies.

**Spec:** `docs/superpowers/specs/2026-07-11-unified-plan-engine-design.md` ‚Äî read it before starting any task.

## Global Constraints

- **Windows:** TRAIN = 2020-01-01..2023-12-31; VALIDATION = 2024-01-01..2025-12-31. Tune on TRAIN only. VALIDATION is run **once per component** (exit model v2, confluence gates, each rescue strategy) with pre-registered gates; results reported as-is, never retuned after.
- **Acceptance gates (unchanged from round 1):** `win_rate >= 80`, `expectancy_r > 0`, `N >= 30` (train) / `N >= 15` (validation), `scratches+timeouts <= 50%` of closed trades.
- **Exit constants:** `STRATEGY_RR_OVERRIDE` values and the 0.30 R:R hard floor are frozen (they ARE the validated TP1). `BREAKEVEN_TRIGGER_FRACTION = 0.5` frozen. `tp1_fraction = 0.50` fixed by spec. `trail_atr_mult` default 2.5, tunable on TRAIN only (Task 30).
- **Win definition:** TP1 touched = win. `win_rate` over win+loss only; `expectancy_r` over all closed trades. Same-bar conservative ordering: stop before target.
- **Weak strategies are never suppressed** ‚Äî they emit plans with the ‚öÝÔ∏è WEAK caution block (user requirement).
- **No ML in the live path.** The logistic-regression audit is offline-only (`scripts/`), never imported by `swingbot/`.
- **Feature flags default off:** `PLAN_ENGINE_V2` (off|shadow|on), `SCALE_OUT_ENABLED`, `INTRADAY_MANAGER_V2`.

- **Backtest data:** cached CSVs via `python scripts/fetch_backtest_data.py` (must be run once before any backtest task; network required).
- **Strategy names (exact strings, used as registry keys):** `EMA Crossover, VWAP, Fibonacci, Support/Resistance, RSI, MACD, Elliott Wave, MA Ribbon, Break & Retest, RSI Divergence, Volume Profile` (see `backtest.py:ALL_STRATEGIES`).
- **DataFrame convention:** daily OHLCV with columns `Open, High, Low, Close, Volume`, DatetimeIndex ‚Äî match `tests/conftest.py` fixtures; read it before writing new tests.

```python
# tests/helpers.py
import pandas as pd

def make_ohlcv(closes, *, start="2024-01-02", spread=0.01, volume=1_000_000):
    """Build a daily OHLCV frame from floats or (open, high, low, close) tuples."""
    rows = []
    for c in closes:
        if isinstance(c, (tuple, list)):
            o, h, l, cl = (float(x) for x in c)
        else:
            o = cl = float(c)
            h, l = cl * (1 + spread), cl * (1 - spread)
        rows.append((o, h, l, cl, float(volume)))
    idx = pd.bdate_range(start=start, periods=len(rows))
    return pd.DataFrame(rows, columns=["Open", "High", "Low", "Close", "Volume"], index=idx)
```

```python
# swingbot/core/plan_engine.py
"""Unified trade-plan engine v2.

Single authority for plan construction and exit policy (spec:
docs/superpowers/specs/2026-07-11-unified-plan-engine-design.md).
"""
from __future__ import annotations
from dataclasses import dataclass, field


class PlanStatus:
    PENDING = "PENDING"
    ACTIVE = "ACTIVE"
    PARTIAL = "PARTIAL"
    CLOSED = "CLOSED"
    CANCELLED = "CANCELLED"


_LEGAL = {
    PlanStatus.PENDING: {PlanStatus.ACTIVE, PlanStatus.CANCELLED},
    PlanStatus.ACTIVE: {PlanStatus.PARTIAL, PlanStatus.CLOSED},
    PlanStatus.PARTIAL: {PlanStatus.CLOSED},
}


@dataclass
class TradePlanV2:
    plan_id: str
    ticker: str
    created_at: str            # ISO date of the bar/scan that created the plan
    source: str                # "strategy" | "confluence"
    strategy: str              # real generating strategy (exact ALL_STRATEGIES string)
    horizon_key: str
    direction: str             # "bullish" | "bearish"
    entry_type: str            # "stop_entry" | "market"
    trigger_price: float
    entry_price: float | None
    expiry_bars: int
    stop_loss: float
    tp1: float
    tp1_fraction: float
    tp2: float | None
    breakeven_trigger_fraction: float
    trail_atr_mult: float
    quality_score: int
    quality_breakdown: list
    tier: str                  # "A" | "B" | "C"
    badge: str                 # "VALIDATED" | "WEAK"
    badge_stats: dict
    status: str
    status_history: list = field(default_factory=list)


def record_transition(plan: TradePlanV2, new_status: str, reason: str | None = None,
                      at: str | None = None) -> None:
    allowed = _LEGAL.get(plan.status, set())
    if new_status not in allowed:
        raise ValueError(f"illegal transition {plan.status} -> {new_status}")
    plan.status = new_status
    plan.status_history.append({"status": new_status, "reason": reason, "at": at})
```

```python
# swingbot/core/registry.py
"""Loader for the committed out-of-sample validation registry.

The JSON is GENERATED by scripts/run_backtest_range.py --emit-registry
(Task 4); hand-edits are forbidden except the initial round-1 seed.
"""
from __future__ import annotations
import json
from dataclasses import dataclass
from pathlib import Path

_PATH = Path(__file__).with_name("validation_registry.json")
_CACHE: list | None = None


@dataclass
class Badge:
    status: str          # "VALIDATED" | "WEAK"
    n: int
    win_rate: float
    expectancy_r: float
    window: str = ""
    run_date: str = ""


def load_registry(path: Path | None = None) -> list:
    global _CACHE
    if _CACHE is None or path is not None:
        _CACHE = json.loads((path or _PATH).read_text(encoding="utf-8"))
    return _CACHE


def reload_registry() -> None:
    global _CACHE
    _CACHE = None


def get_badge(source: str, strategy: str, horizon_key: str | None = None) -> Badge:
    records = load_registry()
    def _find(hz):
        for r in records:
            if r["source"] == source and r["strategy"] == strategy and r.get("horizon") == hz:
                return r
        return None
    rec = _find(horizon_key) or _find(None)
    if rec is None:
        return Badge(status="WEAK", n=0, win_rate=0.0, expectancy_r=0.0)
    return Badge(status=rec["status"], n=rec["n"], win_rate=rec["win_rate"],
                 expectancy_r=rec["expectancy_r"], window=rec.get("window", ""),
                 run_date=rec.get("run_date", ""))
```

```python
def build_registry_records(summaries, *, source, window, run_date,
                           horizon=None, pass_wr=80.0, min_n=15):
    recs = []
    for s in summaries:
        validated = (s["win_rate"] >= pass_wr and s["expectancy_r"] > 0
                     and s["n"] >= min_n)
        recs.append({"source": source, "strategy": s["strategy"], "horizon": horizon,
                     "status": "VALIDATED" if validated else "WEAK",
                     "n": s["n"], "win_rate": round(s["win_rate"], 1),
                     "expectancy_r": round(s["expectancy_r"], 3),
                     "window": window, "run_date": run_date})
    return recs
```

```python
def adx_series(df, period=14):
    h, l, c = df["High"], df["Low"], df["Close"]
    up, dn = h.diff(), -l.diff()
    plus_dm = ((up > dn) & (up > 0)) * up
    minus_dm = ((dn > up) & (dn > 0)) * dn
    tr = pd.concat([h - l, (h - c.shift()).abs(), (l - c.shift()).abs()], axis=1).max(axis=1)
    atr = tr.ewm(alpha=1 / period, adjust=False).mean()
    plus_di = 100 * plus_dm.ewm(alpha=1 / period, adjust=False).mean() / atr
    minus_di = 100 * minus_dm.ewm(alpha=1 / period, adjust=False).mean() / atr
    dx = 100 * (plus_di - minus_di).abs() / (plus_di + minus_di).replace(0, pd.NA)
    return dx.ewm(alpha=1 / period, adjust=False).mean()
```
